Ход игрока¶

Каждый игрок делает запрос «ход текущего игрока».

Запрос к серверу:

{
    "move": 5, // номер текущего хода
    "data": {
        "type": "fold|check|call|raise|all-in", // тип хода
        "bet": 250 // указывается только если тип хода "raise"
    }
}

Ответ от сервера:

{
    "move": 5, // номер текущего хода
    "data": {
        "bank": {
            "total": 75, // общая ставка за все ходы
            "current": 50 // ставка за текущий этап игры
        },
        "board": {
            "cards": [ // карты на столе
                "S6",
                "CJ",
                "SA"
            ]
        },
        "players": [
            {
                "user": {
                    "id": 345345,
                    "chips": 50000, // всего фишек у данного пользователя в игре
                    "place": 0 // место за столом
                },
                "game": {
                    "bet": 50, // ставка игрока за данный этап игры
                    "isDealer": true, // (* указывается только если) пользователь является дилером в данной игре
                    "typeMove": "raise" // (* указывается только если) игрок сделал ход, тип хода
                }
            },
            {
                "user": {
                    "id": 456455,
                    "chips": 38000,
                    "place": 1
                },
                "game": {
                    "bet": 0
                }
            },
            {
                "user": {
                    "id": 145344,
                    "chips": 43000,
                    "place": 2
                },
                "game": {
                    "bet": 0
                }
            }
        ],
        "game": {
            "move": 456455, // id пользователя, чей ход следующий. (* указывается только если) не указан параметр "isFinish"
            "prevMove": {
                "userId": 456455, // id пользователя предыдущего игрока
                "typeMove": "call" // тип хода предыдущего игрока
            },
            "isChangedStage": true, // (* указывается только если) произошла смена этапа игры
            "isFinish": true // (* указывается только если) это был последний ход в данной игре
        }
    }
}

Все игроки ходят по кругу.

Если игрок сбросил карты flop, то в ответе от сервера в data.players его уже не будет. Информация по нему будет только в инфе о ходе предыдущего игрока data.game.prevMove.

Когда последний игрок сделает ход - новый этап игры:

  • выкладываются новые карты на стол (data.board.cards)

  • фишки текущего этапа data.bank.current добавляются в общий банк data.bank.total

  • задаётся флаг о смене этапа игры (data.game.isChangedStage)

Если в ответе есть data.game.isFinish, то данный ход был последним.
