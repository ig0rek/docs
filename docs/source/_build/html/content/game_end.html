Конец игры¶

Сразу после последнего хода в центрифугу прилетит сообщение о завершении игры с результатами.

Ответ от сервера:

{
    "data": {
        "board": {
            "cards": [
                "S6",
                "CJ",
                "SA",
                "H10",
                "DK"
            ]
        },
        "players": [
            {
                "user": {
                    "id": 345345,
                    "chips": 50000,
                    "place": 0
                },
                "award": 125, // (* указывается только если) игрок победил и получил фишки
                "cards": [ // (* указывается только если) все игроки дошли до конца игры и выбор победителя был по результатам карт
                    "CA",
                    "HA"
                ],
                "combination": [ // (* указывается только если) все игроки дошли до конца игры и выбор победителя был по результатам карт
                    "CA",
                    "SA",
                    "HA",

                    "CJ"
                ]
            },
