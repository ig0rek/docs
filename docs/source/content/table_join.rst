
Присоединение к столу
====================================

Игрок делает запрос на "присоединение к столу" и указывает количество фишек, с которыми садится.

.. code-block:: js

    {
        "cash": "#int#",
    }

После того, как второй игрок садится, сервер делает отсчёт в 5 секунд и запускает игру.
